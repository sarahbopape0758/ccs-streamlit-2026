##  Project Overview

This project is an interactive **Cybersecurity Portfolio Web Application** developed using **Python and Streamlit**.

The application functions as a digital CV and a practical demonstration of my programming and cybersecurity knowledge. Instead of a normal static portfolio website, users can interact with tools, encryption demos, and features that show my technical skills in action.

The portfolio presents:

* My education and experience
* Cybersecurity knowledge
* Projects and demonstrations
* Certifications
* Interactive tools
* Downloadable CV

---

## Purpose of the Project

The purpose of this project is to:

* Build a professional online portfolio
* Demonstrate Python programming skills
* Apply cybersecurity concepts practically
* Create an interactive CV for recruiters and lecturers
* Practice UI design and user interaction using Streamlit

## Application Features

### Home

Introduces who I am and what the portfolio contains.
Includes theme switching (Dark Mode / Light Mode).

### About

Displays:

* Education history
* University modules
* Work experience (Tutor & IEC Digital Assistant)

### Projects

Includes:

* Smart Lecturer Scheduling & Navigation System
* Encryption / Decryption demonstration

Users can type a message and see it encrypted and decrypted using a Caesar Cipher.

### Skills

Shows my technical competencies:

* Python
* Java
* C++
* Networking
* Linux
* Cybersecurity fundamentals

### CV Download

Visitors can directly download my CV from the app.

### Ask Me (Chatbot)

A built-in chatbot answers questions about:

* My education
* Skills
* Background
* Studies

### Cyber Card Match Game

A small guessing game where users try to match the AI’s card number.

### Cybersecurity Tool

**SHA256 Hash Generator**

* Demonstrates password hashing
* Demonstrates data integrity
* Shows real-world cybersecurity usage

### Timeline

Displays my journey from matric to final-year university student.

###  Certificates & Badges

The application includes a certificate viewer.
Users can browse through my certificates using **Next** and **Previous** buttons.

##  How to Run the Project Locally

### 1. Clone the repository

```bash
git clone https://github.com/your-username/css2026-sarah-bopape-portfolio.git
```

### 2. Open the project folder

```bash
cd css2026-sarah-bopape-portfolio
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the application

```bash
streamlit run app.py
```
